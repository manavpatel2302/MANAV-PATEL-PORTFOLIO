## Getting Started

Follow these instructions to set up and view the portfolio locally:

### Clone the Repository
```bash
git clone https://github.com/manavpatel2302/MANAV-PATEL-PORTFOLIO
cd MANAV-PATEL-PORTFOLIO
```

## Installation

No additional dependencies are required beyond a modern web browser. For advanced customization, ensure you have [Node.js](https://nodejs.org/) installed for SCSS compilation or JavaScript bundling (if desired).

## Usage

To view the portfolio:

1. **Basic:**  
   Open `index.html` with any web browser.

2. **Development/Customization:**  
   - Edit `styles.scss` for custom SCSS and compile to CSS as needed.
   - Modify `main.js` for interactive features.
   - Replace images in the repository (e.g., `IMG_5912_20250413111751.jpg`) with your own for a personalized look.

## Structure

- `index.html` — Main page layout
- `styles.css`, `styles.scss` — Styling files
- `main.js` — Interactive behaviors and DOM manipulation
- Image assets — For profile, skills, and travel sections
